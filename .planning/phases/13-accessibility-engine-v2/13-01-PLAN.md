# Phase 13 Plan 01

## Goal

Implement the accessibility engine v2 slice:
- Persistent bionic reading, visual pacing, line focus, letter spacing, and word spacing preferences.
- Reading-mode rendering on assignment descriptions, reading panels, note bodies, and transcripts.
- Expanded TTS preferences for provider, voice, speed, and pitch.
- ElevenLabs-compatible TTS Edge Function path.
- Keyboard and ARIA improvements for accessibility controls.
- High-contrast calm-error token verification.

## Acceptance Criteria

- Devon can enable bionic reading and line focus from Settings with one control tap each.
- TTS reads assignment descriptions, note bodies, and cleaned note transcripts aloud when enabled.
- Reading controls and settings controls are reachable by keyboard only.
- Phase 13 schema is applied to the linked Supabase project.
- Static gates remain green: typecheck, Vitest, tone-audit, production build.
