# Phase 34 Plan - Social + Collaboration Features

## Goal

Deliver F169-F173: opt-in study groups, shared flashcard decks, collaborative notes, peer explanation mode, and group project coordination.

## Scope

- Add invite-only study group schema with owner/member RLS and membership helper functions.
- Add shared Pomodoro sessions for group focus blocks.
- Add shared flashcard deck creation and copy shared cards into every member's FSRS queue.
- Add collaborative notes with optimistic version checks and a 500 ms refresh cadence.
- Add peer explanations for Feynman-style concept teaching.
- Add group project tasks with neutral statuses and no ranking or leaderboard surfaces.
- Add `/study-groups` as the authenticated collaboration workspace and expose it in navigation.

## Acceptance Mapping

- F169 Study groups: `/study-groups` creates invite-only rooms with shared Pomodoro sessions and member count.
- F170 Shared flashcard decks: shared cards are copied into member `flashcards` rows through `install_shared_deck_for_members`.
- F171 Collaborative notes: group notes save with version conflict checks and refresh every `500` ms.
- F172 Peer explanation mode: members can add concept explanations in their own words.
- F173 Group project coordinator: group tasks track owner, due date, and neutral status.
- Privacy: groups are invite-only, RLS gates group data by membership, and no ranking/leaderboard/streak surface is introduced.
