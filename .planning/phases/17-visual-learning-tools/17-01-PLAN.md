# Phase 17 Plan 01

## Goal

Implement Visual Learning Tools:
- Mind maps from note content.
- Concept relationship graphs.
- Timeline builder from note events.
- Diagram annotation from uploaded images.
- Whiteboard-style visual study surfaces already introduced in Phase 16.
- Color-coded outline view.
- Comparison table builder.

## Acceptance Criteria

- Note detail has one visual learning panel.
- Mind map and concept graph render as visual networks.
- Timeline renders as a horizontal visual sequence.
- Diagram annotation accepts an uploaded image and overlays labels.
- Color outline renders from existing note outlines without an AI call.
- Comparison table renders from note concepts.
- `visual-tools` is deployed and ACTIVE in Supabase.
- Static gates remain green: typecheck, Vitest, tone-audit, production build.
