# Phase 14 Plan 01

## Goal

Implement Smart Notes v2:
- Cross-note synthesis with source-note citations.
- Automatic related-note links on note detail.
- Full-text note search with snippet previews.
- Outline view continuity from generated note outlines.
- One-tap flashcard creation from highlighted note text.
- Student-set tags and AI-suggested tags.

## Acceptance Criteria

- A synthesis query returns a concise summary with citations that link back to source notes.
- Related notes render on note detail using local similarity scoring.
- Search filters notes and shows a relevant snippet.
- Highlighting note text enables one-tap flashcard creation.
- Tags can be added/removed and AI suggestions can be accepted.
- Static gates remain green: typecheck, Vitest, tone-audit, production build.
