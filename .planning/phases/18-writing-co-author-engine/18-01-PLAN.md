# Phase 18 Plan 01

## Goal

Implement the Writing Co-Author Engine:
- Essay scaffold from prompt and draft context.
- Ghost-text co-write suggestions that the student explicitly accepts.
- Transition, evidence, argument, readability, and tone checks.
- Authorship percentage display.
- Citation formatter continuity through the existing Citation Tool.

## Acceptance Criteria

- Essay assignments open a writing studio instead of only a sentence checker.
- Student draft text is always entered first for content-level help.
- Ghost text is short and only becomes part of the draft after student acceptance.
- Evidence finder uses class note context when available.
- Argument/readability/tone checks produce guidance without rewriting the draft.
- Authorship percentage changes when ghost text is accepted.
- `writing-cowrite` is deployed and ACTIVE in Supabase.
- Static gates remain green: typecheck, Vitest, tone-audit, production build.
