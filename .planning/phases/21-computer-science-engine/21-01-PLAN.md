# Phase 21 Plan 01

## Goal

Implement the Computer Science Engine:
- In-browser JavaScript and Python execution.
- Error-to-hint scaffold that asks guiding questions before any code change.
- Algorithm visualizer for bubble sort, binary search, and linked-list traversal.
- Pseudocode-to-code outline bridge.
- Code review comprehension checks.
- Debugging scaffold.
- AP CSP / AP CSA project milestones.

## Acceptance Criteria

- Computer-science assignments show a Coding Scaffold on assignment detail.
- JavaScript runs in a disposable browser Worker with a short timeout.
- Python runs in-browser for intro-CS print, variables, arithmetic, and `for range(...)` loops.
- Error hint mode never pastes a complete corrected solution.
- Algorithm visualizer works for bubble sort, binary search, and linked-list traversal.
- Pseudocode, code review, debug log, and project scaffold modes return structured prompts.
- The Edge Function enforces red/yellow AI mode restrictions before content-generating help.
- `cs-scaffold` is deployed and ACTIVE in Supabase.
- Static gates remain green: typecheck, Vitest, tone-audit, diff check, production build.
