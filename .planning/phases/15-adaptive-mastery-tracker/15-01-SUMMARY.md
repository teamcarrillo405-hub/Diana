# Phase 15 Plan 01 Summary

Phase 15 is implemented and applied to the linked Supabase project.

## What Changed

- Added migration `0023_adaptive_mastery_tracker.sql`:
  - `mastery_concepts`
  - `mastery_events`
  - `flashcards.concept_id`
  - RLS policies and indexes for concept maps and mastery event history
- Added deterministic concept seeding from class notes, assignments, rubrics, note tags, and fallback concepts.
- Added class-level Mastery map UI with concept mastery bars, confidence check-ins, review-next guidance, and gap bridge copy.
- Linked note-created flashcards and manual note flashcards to concepts when a source note has class/tag context.
- Updated flashcard review flow so every three correct concept-linked reviews increases mastery by one level, capped at 4.
- Added AI quiz result recording on the mastery panel; quiz ratings update mastery conservatively and append `ai_quiz` events.
- Added PDF export for class mastery reports.
- Added read-only concept confidence to parent share summaries.
- Added pure tests for concept seeding, flashcard mastery math, AI quiz adjustment, gap bridge copy, and PDF generation.

## Acceptance Evidence

- Concept map per class:
  - `app/(app)/classes/[id]/page.tsx` calls `ensureClassConcepts` and inserts missing seeds until at least five concepts exist.
  - `deriveConceptSeeds` uses tags and class text first, then safe fallback concepts.
- Flashcard-driven mastery:
  - `app/(app)/flashcards/[id]/actions.ts` counts correct reviews for the linked concept.
  - `masteryLevelFromCorrectReviews` maps three correct reviews to one mastery level.
- AI quiz and self-confidence inputs:
  - `recordAiQuizMasteryResult` writes `mastery_events.source = "ai_quiz"` and moves `mastery_level` toward the quiz rating.
  - `updateConceptConfidence` writes `mastery_events.source = "self_confidence"` and stores `self_confidence` separately.
- Review-next and gap bridge:
  - The class page sorts concepts by lowest mastery for the next review suggestion.
  - `gapBridgeSuggestion` connects the review concept to the strongest concept when available.
- Teacher-safe PDF export:
  - `app/(app)/classes/[id]/mastery/export/route.ts` returns a PDF generated by `buildMasteryReportPdf`.
  - The PDF includes concept mastery/confidence and assignment completion status.
- Parent read-only dashboard:
  - `app/share/[token]/page.tsx` adds the lowest-mastery concepts to the parent summary.
  - `app/share/[token]/parent-summary.tsx` renders a read-only Concept confidence section.
