# Phase 15 Plan 01

## Goal

Implement the Adaptive Mastery Tracker:
- Per-class concept map with mastery levels from 0 to 4.
- Mastery updates from flashcard performance, AI quiz results, and self-reported confidence.
- Shame-free review-next and gap bridge guidance.
- Teacher-safe PDF export with assignment completion and concept confidence.
- Parent read-only summary surface that includes concept confidence.

## Acceptance Criteria

- Each class renders at least five concept seeds from class notes, assignments, rubrics, tags, or safe fallback concepts.
- Mastery level increases after three correct flashcard reviews linked to a concept.
- AI quiz result events update concept mastery and append an audit event.
- Student confidence check-ins persist without changing the mastery score.
- The class page surfaces the lowest-mastery concept and a bridge from a stronger concept when one exists.
- PDF export downloads a valid PDF with concept confidence and assignment completion.
- Parent share summaries include read-only concept confidence.
- Static gates remain green: typecheck, Vitest, tone-audit, production build.
