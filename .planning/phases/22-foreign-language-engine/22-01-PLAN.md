# Phase 22 Plan 01

## Goal

Implement the Foreign Language Engine:
- Vocabulary introduction with cognate hints and interest-based sentences.
- Guided conjugation table.
- Reading comprehension prompts.
- Speaking practice from speech-to-text transcript.
- Target-language writing scaffold.
- Cultural context cards.
- Vocabulary flashcards through the existing FSRS deck with audio playback in review.

## Acceptance Criteria

- Foreign-language assignments show a Language Scaffold on assignment detail.
- Vocabulary cards include meaning, cognate hint, interest-based sentence, and pronunciation text.
- Conjugation mode reveals table rows incrementally.
- Reading mode gives English questions and target-language answer frames.
- Speaking mode accepts speech-to-text transcript and returns nongraded practice prompts.
- Writing mode helps the student compose in the target language without writing the full response.
- Culture mode ties vocabulary to cultural context.
- Vocabulary cards can be saved into the existing FSRS flashcard deck.
- Flashcard review exposes TTS audio controls using the student's saved provider, including ElevenLabs when configured.
- `language-scaffold` is deployed and ACTIVE in Supabase.
- Static gates remain green: typecheck, Vitest, tone-audit, diff check, production build.
