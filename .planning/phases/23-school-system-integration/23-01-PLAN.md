# Phase 23 Plan 01 - School System Integration

## Scope

Deliver the v2 school-system integration substrate:

- Canvas OAuth connection, assignment import, due dates, original assignment links, and rubric text.
- Google Classroom assignment import plus announcement capture into the Diana inbox.
- Clever school-managed marker row for district-provisioned setup.
- IEP / 504 text import that applies extended time, TTS, dyslexia font, font size, and line spacing preferences.
- Background sync entrypoint that refreshes stale Canvas, Classroom, and ICS connections from Settings.
- External submission handoff tracking for Canvas and Google Classroom assignments.
- Diana due-date `.ics` export for Google Calendar / Apple Calendar subscription or import.

## Implementation Notes

- Migration `0024_school_system_integration.sql` extends `assignments`, permits Clever in LMS checks, and adds `iep_imports`.
- Canvas OAuth requires `CANVAS_CLIENT_ID` and `CANVAS_CLIENT_SECRET` in the deployment environment.
- IEP upload uses browser text extraction / paste. Readable text files work directly; scanned PDFs still need OCR or the existing document extraction path.
- Clever is represented as a district provisioning marker in Phase 23, not full Clever API SSO.
- Submission sync is an explicit student handoff: open the source assignment, submit in Canvas/Classroom, then mark the handoff in Diana.
- Background sync is app-triggered when Settings opens and a connection is stale; no cron or service worker scheduler is added in this phase.

## Acceptance Mapping

- F96 Canvas LMS OAuth: `app/api/lms/canvas-oauth/start`, `app/api/lms/canvas-oauth/callback`, `lib/lms/canvas.ts`.
- F97 Google Classroom OAuth: existing Google session token path, `app/api/lms/classroom-sync/route.ts`, Classroom announcements into `inbox_items`.
- F98 Clever SSO: `connectClever` marker and provider constraint support.
- F99 IEP / 504 import: `lib/iep/import.ts`, Settings import panel, `iep_imports` audit table.
- F100 Assignment auto-sync: `app/api/lms/sync-all/route.ts` plus stale-connection Settings trigger.
- F101 Submission sync: `ExternalSubmissionSync` plus `markExternalSubmission`.
- F102 Calendar sync: `lib/lms/export-ics.ts` and `app/api/calendar.ics/route.ts`.
