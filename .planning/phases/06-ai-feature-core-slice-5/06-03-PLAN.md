<objective>
Ship the three Phase 6 AI Edge Functions (F09 math-step, F10 writing-aid, F11
citation-gen). Every Edge Function in this plan must:

1. Read `aiMode` from the request body; return 403 if `'red'`; for `'yellow'` allow
   only `citation-gen` (math + writing return 403 on yellow).
2. Call `resetBudgetIfNewDay` → `checkTokenBudget`; return 429 with a calm message
   when the per-user token budget is exhausted.
3. Compose the system prompt using `composeSystemPrompt` from the shared module,
   injecting the F17 refuse-with-redirect fragment and (for math + writing) the
   F18 frustration redirect.
4. Call Anthropic, capture `usage.input_tokens + usage.output_tokens`, log to
   `ai_interactions`, and increment `profiles.tokens_used_today`.

This plan runs PARALLEL to Plan 06-02 (both are wave 2 — they share no files). Both
depend on 06-01 for the schema + shared safety modules. The server actions, client
components, and assignment detail page wiring that consume these Edge Functions land
in Plan 06-04 (wave 3, depends on 06-01 + 06-02 + 06-03).

Output:
- Three new Edge Functions under `supabase/functions/` — and nothing else. No app/
  files, no client components. Plan 06-04 handles all UI consumption.
</objective>
