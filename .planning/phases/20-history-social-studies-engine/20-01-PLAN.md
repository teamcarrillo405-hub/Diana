# Phase 20 Plan 01

## Goal

Implement the History & Social Studies Engine:
- Primary source annotation prompts from pasted or uploaded source text.
- Cause/effect chain scaffolding.
- HAPP source analysis.
- DBQ outline scaffolding.
- Compare/contrast table scaffolding.
- Map image annotation with student label quiz prompts.
- Current-events connection prompts.

## Acceptance Criteria

- History/social-studies assignments show a History Helper on assignment detail.
- Text-like source files can populate the source field without a server round trip.
- The helper supports primary source, cause/effect, HAPP, DBQ, compare/contrast, and current-events modes.
- Map image upload stores to the existing `note-docs` bucket and returns positioned annotation prompts.
- HAPP output covers Historical context, Audience, Purpose, and Point of view.
- DBQ mode returns a six-part outline from the supplied prompt/documents.
- The Edge Function enforces red/yellow AI mode restrictions before content-generating help.
- `history-scaffold` is deployed and ACTIVE in Supabase.
- Static gates remain green: typecheck, Vitest, tone-audit, diff check, production build.
